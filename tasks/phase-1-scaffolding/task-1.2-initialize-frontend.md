# Task 1.2 — Initialize Frontend Project

## Description
Scaffold the React + TypeScript frontend with Vite.

## Status
Complete

## Subtasks
- Run `npm create vite@latest frontend -- --template react-ts`
- Install Tailwind CSS and configure `tailwind.config.ts`
- Install and configure shadcn/ui
- Install React Router v6
- Install TanStack Query
- Install Zustand
- Create basic layout component with sidebar navigation
- Set up API client base (axios or fetch wrapper)

## Dependencies
None

## Acceptance Criteria
- `npm run dev` starts the dev server
- Tailwind styles apply correctly
- shadcn/ui components render
- Router renders placeholder pages
