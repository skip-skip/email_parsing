# Task 12.2 — LLM-Generated Drafts via EmailDraftAgent

## Description
The `_create_missing_info_entry()` method in `EmailProcessor` currently generates missing info drafts using a hardcoded template. Replace this with the `EmailDraftAgent` which uses the LLM to generate higher-quality, personalized drafts that reference the specific email context.

## Status
Not Started

  - Draft is correctly added to the queue
